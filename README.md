#Instructions.

- Build project and create artifact (a jar with dependencies in same level)
- Create a folder called files in the same level of the jar.
- Create a folder inside files called `input`. Folder structure will now be ./files/input/
- Put images to classify in the `input` folder.
- launch app and start classification.
- Output will be placed in ./files/output/valid/ for valid images and ./file/output/invalid/ for invalid images.
